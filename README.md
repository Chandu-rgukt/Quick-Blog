# QuickBlog - Modern Blogging Platform

A full-stack MERN blog application with AI-powered content generation, built with React, Node.js, Express, MongoDB, and Google Gemini AI.

## 🚀 Features

- **AI-Powered Content Generation**: Integrated with Google Gemini AI for automatic blog content creation
- **Rich Text Editor**: Quill.js based editor for beautiful blog content
- **Image Optimization**: Automatic image optimization and WebP conversion with ImageKit.io
- **Admin Dashboard**: Complete admin panel for content management
- **Comment System**: User comments with admin moderation
- **Responsive Design**: Mobile-first responsive design
- **Category Filtering**: Organize blogs by categories
- **Search Functionality**: Real-time search across blog content
- **Authentication**: JWT-based admin authentication

## 🛠️ Tech Stack

### Frontend
- **React 18** with React Router DOM
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Quill.js** for rich text editing
- **React Hot Toast** for notifications
- **Axios** for API calls

### Backend
- **Node.js** with Express.js
- **MongoDB** with Mongoose ODM
- **JWT** for authentication
- **Multer** for file uploads
- **ImageKit** for image optimization
- **Google Gemini AI** for content generation
